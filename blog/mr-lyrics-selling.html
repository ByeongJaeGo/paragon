PARAGON 음악 작품 마켓 이용 가이드

안녕하세요. 오늘은 기획사나 레이블 컨택 없이 작곡 MR이나 가사 작품을 올리고 판매할 수 있는 방법을 정리해 봤습니다.

많은 작곡가·작사가 "작품은 있는데 낼 데가 없다"고 하죠. 저도 그래서 PARAGON이라는 작품 마켓을 만들었습니다.

https://paragon-inky.vercel.app

PARAGON이란?

작곡(MR/WAV)과 작사(가사 문서) 작품을 올리고 가격을 정해 판매할 수 있는 웹 마켓입니다.

  • 프로필 만들기 (닉네임, 장르, 역할)
  • 작품 올리기 + 가격 설정
  • 마켓에서 구매·문의

2단계면 시작할 수 있습니다.

작곡 MR 판매하는 방법

  1. PARAGON 접속
  2. [프로필 만들기] — 역할에서 작곡 선택
  3. [작품 올리기] — 미디작곡 선택
  4. 제목, BPM, WAV 파일, 판매 가격 입력
  5. 업로드 완료 → 마켓에 노출

가사·작사 작품 판매하는 방법

  1. 프로필 — 역할 작사 선택
  2. [작품 올리기] — 작사 선택
  3. 가사 문서(TXT 권장) + 핵심 한 줄(맛보기) + 가격
  4. 업로드 → 구매 전 맛보기, 구매 후 전체 확인

지금은 베타 테스트 중

서비스를 키워가는 중이라 작곡·작사 하시는 분들의 피드백을 받고 있습니다. 한번 등록해 보시고 의견 남겨 주시면 감사합니다.
